
MILLETS MOMO - ORDER TRACKER
AI CODING AGENT IMPLEMENTATION BRIEF (MASTER SPECIFICATION)

PURPOSE
Build a production-ready Progressive Web App (PWA) for Millets Momo, a food cart business. The application is an internal order tracking system used by Staff and Admin users. The generated application must closely follow the supplied PRD and preserve the original prototype design while elevating quality, responsiveness, accessibility, animations, and overall UX polish.

================================================================================
1. PRODUCT OVERVIEW
================================================================================

Application Type:
- Progressive Web App (PWA)
- Installable on Android and iOS
- Responsive for phone, tablet, and desktop

Business Purpose:
- Track daily customer orders
- Record payment status
- Monitor sales and revenue
- Provide daily reporting and analytics

Personas:

1. Staff
   - Login via 4-digit PIN
   - Select business date
   - Create and manage orders
   - Mark orders complete
   - Resolve pending payments

2. Admin
   - Login via separate PIN
   - Access reporting dashboard
   - Review orders
   - View revenue and payment analytics

================================================================================

- Order must contain at least one item
- Quantity >= 1

- Pending orders require payment resolution before completion
- Delete order requires confirmation
- Historical prices must remain preserved
